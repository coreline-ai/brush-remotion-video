# 심해의 빛 — 60씬 이미지 생성 규격

## 공통 이미지 규격

- 목적: 60씬 브러시 리빌 영상용 배경 이미지
- 화면: 16:9 가로 풀 프레임, 최종 1920×1080
- 매체: 반실사 수묵채색화, 눈에 보이는 붓결과 번짐
- 배경: 어두운 먹빛·남청색, 청록·보라·생물발광 포인트
- 구성: 한 씬 한 중심 피사체, 피사체 전체가 화면 안에 들어오는 넓은 구도

## 금지 항목

- 순수 실사 사진처럼 보이는 렌더
- 주요 피사체 잘림, 화면 밖 이탈, 과도한 클로즈업
- 레터박스·필러박스·검은 테두리·콜라주
- 텍스트·로고·워터마크
- 동일한 생물·구도 반복
- 브러시가 그려질 영역을 가리는 별도 그래픽 오버레이

## 생성 후 QA

1. 01~60 파일 누락 여부 확인
2. 이미지 실제 해상도와 종횡비 확인
3. 풀 화면 커버리지 확인
4. 주제와 이미지의 의미 일치 확인
5. 인접 씬 중복·동일 구도 확인
6. 반실사 수묵채색화 스타일 편차 확인
7. 통과한 이미지에만 브러시 라우트를 연결

브러싱 로직은 기존 승인 버전 `cosmic-random-brush`를 유지하고, 이번 작업에서는 이미지 에셋만 교체한다.
