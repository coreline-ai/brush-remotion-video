# 우주 풍경 60씬 풀스크린 이미지 마스터 스펙

이 문서는 이후 이미지 검색·생성·합성·QA의 단일 기준이다. 제목이 비슷해도 아래의 필수 피사체와 구도를 만족하지 못하면 사용하지 않는다.

## 전역 규칙

- 최종 이미지는 모두 `1920×1080`, 진짜 16:9 풀블리드다.
- 레터박스, 필러박스, 중앙 패널, 검은 사각형, 흐린 복제 배경을 허용하지 않는다.
- 핵심 피사체 전체를 프레임 안에 두고 가장자리 안전 여백을 최소 6% 확보한다.
- 원본 비율이 맞지 않으면 중앙 크롭하지 않는다. 16:9 원본으로 교체하거나 자연스럽게 아웃페인팅한다.
- 도표, 라벨, 범례, 콜라주, 관측 스트립, UI, 워터마크를 허용하지 않는다.
- 동일 원본·유사 구도를 다른 씬에 재사용하지 않는다.
- 이미지 제목보다 아래의 `필수 화면`과 `금지` 조건을 우선한다.

## 제작 게이트

1. 이미지 제작 전에 scene 번호·확정 주제·필수 화면을 prompt와 manifest에 그대로 기록한다.

3. 한 항목이라도 실패하면 보정하지 않고 원본 교체 또는 재생성한다.
4. 먼저 01·27·34번으로 지구 풍경·행성 합성·성운 확장의 기준을 승인받는다.
