# cosmic-random-brush v0.4 — 60씬 이미지 에셋 전면 재정리

작성 일시: `2026-07-12 16:42:32 KST`

## 문제

- v0.3 에셋 수집기가 검색 첫 결과를 자동 채택해 도표·콜라주·관측 스트립도 통과했다.
- 대부분의 원본을 중앙 `cover` 크롭해 행성·위성·세로 이미지가 잘렸다.
- 기존 QA는 1920×1080·해시·중복·브러시 coverage만 검사했고, 피사체의 완전성·식별성은 검사하지 않았다.

## 목표

- 60장 모두 피사체가 한눈에 식별되는 1920×1080 다크 우주 이미지로 재구성한다.
- 16:9 풍경은 safe-cover, 행성·원형 피사체는 전체가 보이는 dark-stage 합성을 적용한다.
- 도표·텍스트·콜라주·부분 관측 스트립을 제거한다.
- 영상 재렌더 전에 에셋 contact sheet와 HTML 검토 페이지를 먼저 승인받는다.
