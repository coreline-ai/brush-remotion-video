# implement_20260712_134800.md

작성 일시: `2026-07-12 13:48:00 KST`

이 문서는 `cosmic-random-brush-video` v0.1의 승인 브러싱을 바꾸지 않고,
서로 다른 우주 구도 6개에서 범용성을 검증하는 v0.2 작업 범위를 고정한다.
최종 산출물은 10초 씬 6개를 연결한 60초 검증 영상과 씬별 QA 증거다.

## 개발 목적

- 지구 일출 1씬에 고정된 v0.1을 대표 우주 구도 6종으로 검증한다.
- 승인된 자유 랜덤 터치 방식, 붓 폭, 이동 리듬, 99.1% 마스크 계약을 유지한다.
- 화면 전체 coverage와 실제 가시 콘텐츠 coverage를 분리해 측정한다.
- 6개 씬을 개별 평가하면서 하나의 60초 검증 MP4로 렌더한다.
- 6씬이 통과하기 전에는 60씬 제작이나 10분 마스터 작업으로 확장하지 않는다.

## 개발 범위

- 대표 씬: 지구 궤도 일출, 토성 고리, 나선은하, 컬러 성운, 블랙홀·강착원반, 저대비 달 표면.
- 포맷: YouTube 1920×1080, 30fps, 씬당 300프레임, 총 1800프레임/60초.
- 입력: 장면별 이미지 6장. 첫 씬은 v0.1 승인 원본과 seed를 그대로 사용한다.
- 이미지마다 제목·출처·라이선스/사용 근거·전처리 기록을 manifest로 보존한다.
- `cosmic-random-brush` 프로파일은 1씬 골든 모드와 6씬 검증 모드만 허용한다.
- 각 씬은 결정적 seed, 기본 36터치, 동일 폭 보완 1~20터치, coverage 0.991 이상을 유지한다.
- 가시 콘텐츠 영역을 alpha·명도·채도 기반으로 추출해 content coverage 0.985 이상을 검사한다.
- 6씬 QA JSON, 핵심 프레임 36장, 통합 contact sheet, HTML 검토 페이지, video-auditor 보고서를 생성한다.

## 제외 범위

- 60개 이미지 생성, 60씬 렌더, 600초 결합.
- 4K, Shorts, 가변 fps·가변 씬 길이.
- 붓 폭을 365px 이상으로 확대하는 커버리지 보정.
- 의미 윤곽 추적, 오브젝트 순차 페인팅, 수백 개 stamp/seal 터치.
- v0.1 지구 일출 골든 fixture 갱신.
- 신규 브러시 디자인, 브러시 물리 시뮬레이션, Remotion/Zod 경고 해결.
- 장시간 BGM 큐레이션, TTS/STT, 챕터 자막 시스템.
- 기존 brush/pen/pen-brush/shorts 프로파일의 시각 튜닝.
