이 문서는 이전 5곡이 모두 잔잔한 수면 피아노 감성으로 수렴한 문제를 바로잡기 위한 재제작 작업 문서다. 이전 산출물은 보존하되 이번 결과로 간주하지 않는다.

## 개발 목적

- 실제 Noct-Salamander 그랜드피아노 샘플로 10분 원곡 5개를 다시 만든다.
- 곡마다 장르, 박자, 그루브, 화성 어휘, 왼손 패턴, 선율 구조를 의도적으로 다르게 만든다.
- 각 결과물은 `44.1kHz / 16-bit / stereo PCM WAV`와 provenance·기술 QA를 남긴다.

| ID | 제목 | 장르 / 박자 | 핵심 차별점 |

- 이전 `sampled-piano-collection`은 삭제하거나 덮어쓰지 않는다.
- 새 project JSON은 `projects/sampled-piano-genre-collection/`, 새 WAV는 `output/original-audio/sampled-piano-genre-collection/`에만 둔다.
- 모든 음은 로컬 Noct-Salamander WAV/SFZ 샘플로 렌더한다. 완성곡·루프·외부 음원은 사용하지 않는다.

## 공통 진행 규칙

- 각 곡은 600초이며, `genre`, `timeSignature`, `tempoUnit`, `groove`, `harmony`, `form`을 composition에 기록한다.
- 장르 다양성은 제목·템포 차이가 아니라 박자/리듬/화성/구성 차이로 검증한다.
- 렌더 단계는 로컬 샘플만 읽고, master 생성 시에만 resample·loudness 처리를 한다.
- Noct-Salamander CC BY 3.0의 출처표시와 변경 사실을 각 provenance와 YouTube 설명문 후보에 포함한다.
