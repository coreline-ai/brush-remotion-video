## 개발 목적

- `cosmic-random-brush-video`를 `dark-random-brush-video`로 변경해 우주·심해·야간 풍경 등 어두운 16:9 이미지에 재사용한다.
- 기존 골든과 렌더러의 `cosmic-random-brush` runtime 동작은 깨뜨리지 않고, YAML 입력에는 `dark-random-brush`를 권장 alias로 제공한다.
- 랜덤 붓 터치, 99.1% coverage, 풀 화면 에셋, 결정성, 60씬 장기 렌더, BGM·최종 QA에서 보완된 내용을 스킬 계약에 통합한다.

## 개발 범위

1. 스킬 디렉터리·메타데이터·설치 링크·README·director/project-yaml 관련 문서를 `dark-random-brush-video` 중심으로 갱신한다.
2. `drawing.profile: dark-random-brush` 입력을 기존 runtime profile `cosmic-random-brush`로 정규화한다.
3. fidelity contract에 공통 에셋/브러시/coverage/씬/오디오/QA/납품 규칙과 역사적 골든 호환 정책을 기록한다.
4. profile alias 파싱 회귀 테스트와 기존 cosmic profile 회귀 테스트를 실행한다.

## 제외 범위

- 기존 cosmic route geometry, seed, golden fixture, 렌더러 로직의 변경.
- 기존 output/asset 재생성 또는 60씬 영상 재렌더.
