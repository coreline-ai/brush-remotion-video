# implement_20260717_122831.md

작성 일시: `2026-07-17 12:28:31 KST`

이 문서는 기존의 단일 1분 샘플 피아노 데모 계획을 대체하지 않고 확장한다. Noct-Salamander CC BY 멀티샘플과 새 MIDI 작곡으로, 기존 프로젝트 이미지의 분위기를 반영한 10분 피아노 음원 5개를 제작·검증한다.

## 개발 목적

- 전자 합성음이 아닌 로컬 Noct-Salamander 피아노 샘플로 10분짜리 원곡 5개를 만든다.
- 기존 이미지 프로젝트의 시각적 분위기를 선율·조성·템포·다이내믹에 반영한다.
- 각 결과물을 `44.1kHz / 16-bit / stereo PCM WAV`와 provenance로 남긴다.
- CC BY 출처표시와 변경 사실을 모든 결과물에 연결한다.

## 개발 범위

| ID | 이미지 참조 | 제목 | 음악 방향 |
|---|---|---|---|
| `star-seed` | `projects/star-seed-fairy-tale-100s/` | 별씨앗의 밤 | D장조, 58 BPM, 높은 음역의 밝은 응답 선율 |
| `rain-cloud` | `projects/storybook-rain-cloud-100s/` | 창가의 빗구름 | G장조/E단조, 54 BPM, 비어 있는 쉼과 부드러운 하행 선율 |
| `deepsea-light` | `output/deepsea-light-v01-60-final-contact-sheet.jpg` | 심해의 푸른 등불 | E장조/C#단조, 52 BPM, 깊은 왼손과 넓은 간격의 멜로디 |
| `cosmic-ink` | `output/cosmic-random-brush-v05-ink-60-final-contact-sheet.jpg` | 먹빛 성운 | B단조, 56 BPM, 낮은 강도와 느린 질문·응답 구절 |
| `africa-dusk` | `output/africa-pen-60/` | 황금 초원의 저녁 | A장조, 60 BPM, 따뜻한 5음 음계 응답과 안정 종지 |

- 각 곡은 정확히 600초이며, 1분짜리 루프를 반복하지 않는다.
- 각 곡은 별도의 composition JSON, raw 48kHz/24-bit render, 44.1kHz/16-bit master, provenance JSON을 가진다.
- 공통 renderer는 SFZ key/velocity mapping을 읽어 실제 WAV 샘플만 재생한다.

## 공통 진행 규칙

- 모든 곡은 seed·theme·composition SHA-256·instrument archive SHA-256을 provenance에 기록한다.
- 다운로드와 렌더를 분리한다. 렌더 단계는 로컬 샘플 파일만 읽는다.
- master는 44.1kHz/16-bit WAV로 제공하고, 영상 믹스 필요 시에만 기존 48kHz 경로로 변환한다.
- CC BY attribution과 변경 고지를 결과물 metadata 및 YouTube 설명문 후보에 기록한다.
- 각 Phase는 자체 테스트가 통과해야 다음 Phase로 진행한다.
