## 개발 목적

**자막(SRT) 또는 대본만 있으면 Supertonic TTS로 더빙을 자동 생성**하는 옵션을 파이프라인에 추가한다.
스킬은 얇게 유지 — 구현은 brushvid, 스킬은 옵션 문서 + 미설치 시 설치 안내만 담당.

## 개발 범위

- `pipeline/pyproject.toml` — optional extra `tts = ["supertonic"]` (기본 의존성에 넣지 않음)
- `brushvid/tts.py` — 문장 분리(한국어 문장부호) → 문장별 합성(duration 수신) → pause 삽입 wav 연결 + SRT 생성.
  supertonic 미설치 시 **명확한 설치 명령을 담은 에러** (침묵 폴백 금지)
- `brushvid/project.py` 확장 — `input.tts{engine, voice, pauseMs, timing(srt|tts, 기본 tts)}` + `input.script`(대본 텍스트) 지원.
  모드 매트릭스: srt+audio(기존 우선) / **srt+tts** / **script+tts** / audio만(whisper) / 무입력(앰비언트)
- `bin/build.py` — tts 스테이지 통합 (stt 자리에서 분기: tts 설정 시 합성, 이후 스테이지 무변경)
- 타이밍 전략: **TTS duration이 시계** (SRT는 텍스트 소스, 타이밍 재계산). `timing: srt`는 후속 여지로 옵션 자리만
- PoC 산출물: `output/tts-poc/*.wav` 한국어 샘플 3문장 (사용자 청취 판정용) + 합성 속도 실측
- `examples/tts-script/` E2E + 스킬 문서(SKILL.md·project-yaml-guide.md)에 TTS 섹션·설치 가이드

## 제외 범위
