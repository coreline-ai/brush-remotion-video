# implement_20260712_151235.md

작성 일시: `2026-07-12 15:12:35 KST`

이 문서는 대표 6씬에서 PASS한 `cosmic-random-brush-video` v0.2를 60씬·600초 본편으로 확장하는
v0.3 제작 범위를 고정한다. 승인된 자유 랜덤 터치 geometry는 변경하지 않는다.

## 개발 목적

- 우주에서 보는 풍경 60개를 각 10초 랜덤 브러시 씬으로 제작한다.
- 60씬을 6개 챕터로 구성해 단순 이미지 나열이 아닌 10분 흐름을 만든다.
- v0.1 골든과 v0.2 대표 6씬 품질 계약을 유지한다.
- 60개 이미지의 출처·크레딧·SHA-256을 manifest로 관리한다.
- 18,000프레임 렌더와 60씬 QA를 재개 가능한 배치 파이프라인으로 수행한다.

## 개발 범위

- 1920×1080, 30fps, 씬당 300f, 총 18,000f/600.000초.
- 60개 이미지와 60개 제목, base seed `260712`, 씬별 `base + index×37`.
- 기본 36터치 + 동일 폭 보완 1~20터치, 붓 폭 230~365px.
- 전체 coverage 0.991, 가시 콘텐츠 coverage 0.985 이상.
- 로컬 synth BGM 600초, H.264/AAC 48kHz stereo 최종 MP4.
- 전체 routes 수치 QA, 60씬 대표 프레임 QA, 59개 경계 분석, v0.1/v0.2 회귀 검사.

## 제외 범위

- 브러싱 방식·강모 모양·폭 범위 변경.
- 의미 윤곽 추적, 작은 stamp 대량 추가, bitmap fade로 coverage 우회.
- 4K, Shorts, 가변 길이, 내레이션/TTS.
- 60개 AI 이미지 모델 학습이나 신규 이미지 공급자 구현.
- Remotion/Zod 버전 경고 해결과 무관한 리팩터링.

## 6개 챕터 / 60씬

| 챕터 | 씬 | 주제 |
|---|---:|---|
| 1. 푸른 행성의 가장자리 | 01~10 | 지구 일출, 야간 지구, 오로라, 구름, 대륙, 도시 불빛, Earthrise |
| 2. 달과 화성의 지평선 | 11~20 | 달 표면·크레이터·지구돋이, 화성 지평선·사구·협곡·극관·위성 |
| 3. 태양계의 거인들 | 21~30 | 수성, 금성, 목성, 대적점, 이오, 유로파, 토성, 천왕성, 해왕성 |
| 4. 별이 태어나는 구름 | 31~40 | 타란툴라·카리나·오리온·독수리·헬릭스·게·베일 성운과 성단 |
| 5. 은하의 바다 | 41~50 | 우리은하, 안드로메다, 막대나선, 소용돌이, 솜브레로, 충돌·렌즈 은하 |
| 6. 빛 너머의 우주 | 51~60 | 블랙홀, 강착원반, 펄서, 퀘이사, 우주망, 외계행성, 원시별, 심우주 피날레 |

## 공통 진행 규칙

- 60씬 제작 중에도 scene-01 v0.1 source와 geometry hash를 유지한다.
- 이미지 미확보·손상·중복은 대체 후 다음 단계로 넘어간다.
- 전체 60씬 렌더 전에 routes/props/QA 메타를 먼저 검사한다.
- 장시간 QA는 60씬마다 핵심 3프레임을 최종 MP4에서 추출하고 챕터별 고위험 씬은 추가 확인한다.
- 렌더 실패 시 캐시와 `--from`으로 재개하며 처음부터 반복하지 않는다.
- 외부 이미지의 원본 페이지·정확한 크레딧을 manifest에 기록한다.
