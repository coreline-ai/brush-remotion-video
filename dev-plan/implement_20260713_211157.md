## 개발 목적

- 사용자가 촬영 전문 용어를 몰라도 자신의 언어로 영상 장면과 움직임을 설명할 수 있게 한다.
- 사용자 표현을 37개 고유 카메라·시점·렌즈·시간·전환 기법 중 하나 이상의 canonical intent로 정규화한다.
- 방향, 속도, 피사체, 거리, 시작/종료 구도, 초점, 안정화, 전환 연속성, 금지 조건이 포함된 전문 프롬프트를 한국어와 영어로 생성한다.
- 같은 의도를 `remotion-still`, `ai-video`, `image-generation`, `scene-transition` 타깃에 맞게 다르게 설명한다.
- 현재 렌더러가 지원하지 않는 3D·연속 영상 기법을 가능한 것처럼 약속하지 않고 호환성 수준을 명시한다.
- 현재 9개 스킬 구조를 유지하고 자연어 번역 책임이 이미 있는 `brush-director`를 확장한다.

## 핵심 결정

1. **새 스킬이 아니라 `brush-director` 확장**으로 구현한다. 정식 스킬 수는 9개를 유지한다.
2. 카메라 기법은 렌더 기능이 아니라 먼저 **전문 프롬프트 변환 지식 계약**으로 편입한다.
3. 스킬이 직접 실행 코드를 포함하지 않는 현재 원칙을 유지한다. validator만 리포의 `bin/`과 `pipeline/tests/`에 둔다.
4. 모델이 이해하기 쉬운 자유 문장과 자동화가 읽을 수 있는 구조화 결과를 함께 제공한다.
5. 카메라 요청이 없으면 기존 director 출력과 표현을 바꾸지 않는다.
6. 미구현 타깃에는 `external-required` 또는 `composition-only`를 표시하고 실행 가능한 YAML 필드로 위장하지 않는다.

## 출력 계약

기존 director의 `연출 브리프 → project.yaml 초안 → 확인 질문 최대 2개` 순서를 유지한다.
카메라 의도가 있을 때만 연출 브리프와 YAML 사이에 다음 `Camera Prompt Pack`을 추가한다.

```yaml
cameraPrompt:
  userRequest: "그림이 완성되면 천천히 인물에게 다가가 줘"
  interpretation:
    canonicalId: slow-zoom-in
    canonicalNo: 8
    emotionalIntent: 집중
    confidence: 0.94
    needsClarification: false
  parameters:
    direction: inward
    speed: slow
    durationSec: 2.5
    startComposition: medium-wide
    endComposition: medium-close-up
    focalPoint: character-face
    subjectLock: true
    stabilization: cinematic-smooth
    motionBlur: minimal
  promptKo: "..."
  promptEn: "..."
  negativePrompt:
    - no line thickening
    - no softened outlines
    - no subject deformation
    - no abrupt crop
  compatibility:
    remotionStill: supported
    aiVideo: supported
    imageGeneration: composition-only
    sceneTransition: not-applicable
```

- `cameraPrompt`는 전문 브리프용 결과이며 이번 범위에서 `project.yaml`에 그대로 삽입하지 않는다.
- 타깃 미지정 시 model-neutral 한/영 프롬프트와 4개 타깃 compatibility를 모두 제공한다.
- 카메라 의도가 없으면 `Camera Prompt Pack`을 생략하고 현재 3개 출력 계약을 그대로 유지한다.

## 카탈로그 최소 계약

각 canonical technique은 다음 정보를 가진다.

| 필드 | 역할 |
| --- | --- |
| `id`, `canonicalNo` | 안정적인 영문 ID와 원본 번호 |
| `nameKo`, `nameEn` | 한/영 전문 명칭 |
| `category` | camera-motion, tracking, rig, aerial, viewpoint, lens-effect, time-effect, transition |
| `aliases` | 37~45 legacy 번호 및 일반 명칭 |
| `naturalLanguage` | 대표 사용자 표현과 동의어 |
| `emotionalIntents` | 웅장함, 집중, 긴장, 몰입 등 추천 의도 |
| `requiredSlots` | 방향·속도·피사체·시작/종료 구도 등 필수 정보 |
| `defaults` | 문맥으로 보완 가능한 안전 기본값 |
| `conflictsWith` | static+handheld, zoom-in+pull-back 같은 충돌 |
| `compatibility` | 4개 타깃별 supported/simulated/composition-only/external-required/not-applicable |
| `promptKo`, `promptEn` | 전문 문장 조립의 핵심 동사구 |
| `negativeRules` | 변형·흐림·방향 역전·검은 테두리 등 금지 조건 |
