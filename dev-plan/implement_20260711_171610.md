## 개발 목적

붓(수묵 리빌)과 구별되는 **"펜 드로잉" 프로파일** — 펜이 빠르게 선화를 그려가는 화이트보드/스케치 영상 —
을 brush_remotion_video에 편입하고, **프로파일 스킬**(`pen-video`)로 노출한다.
엔진·파이프라인은 하나를 공유한다 (video-builder의 "실험 파일 증식" 전철 방지).

## 개발 범위

- **Phase 1 (근사 검증, 코드 0)**: 펜 커서 PNG 에셋 + pen 프리셋 props 데모 렌더 → **사용자 느낌 판정 게이트**
  - 근사 프리셋: `faint: 1.0`(즉시 또렷), `drawSpeedScale: 0.3~0.5`(빠른 템포), `touchJitter/pathJitter: 0`,
    `edgeFeather: 0`, `developFrames: 8~12`(팝업 방지 최소 마감), `prewashOpacity: 0`, `randomizeOrder: true`, `linearDraw: true`
  - 마스크 폭은 축소하지 않음 — **얇은 펜 선은 배경 line-art가 담당** (video-builder 실측: 펜 마스크 74~132px)
- **Phase 2 (신규 구현)**:
  - `project.yaml`에 `drawing.profile: pen | brush(기본)` 노출 — routes 생성 시 drawEnd를 duration의 30~40%로
    당기는 타이밍 프로파일 (`RouteParams.draw_end` 기지원 — 파이프라인 노출만)
  - `CursorLayer`에 `brush.kind: "pen"` — 내장 벡터 펜 스프라이트 + 펜 전용 회전 상수 (붓의 +180°/기울기 계수와 분리)
  - 선화 배경 프리셋: imagegen 프롬프트 "pen line-art sketch" 변형 (background-prompt.md 확장)
- **Phase 3 (스킬 편입)**:
  - `skill/pen-video/SKILL.md` 신설 — **프로파일 스킬**: pen 프리셋 값 + 선화 배경 가이드만 담고 실행은 `bin/build.py` 공유 (코드 사본 0)
  - brush-director의 mood-presets에 ✒️ 펜 스케치 프리셋 추가
  - `bin/install-skills.sh`에 pen-video 추가 + 설치 + E2E (선화 배경 1씬)
